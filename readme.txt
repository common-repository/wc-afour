== Description ==

This lightweight plugin allows you to enable the Billing and/or Shipping fields to the Customer Registration Form. For this plugin to work you must have customer registration enabled in the WooCommerce settings.

The LITE version includes the options:

* Make billing name fields visible
* Make billing name fields required
* Make billing address fields visible
* Make billing address fields required
* Make shipping name fields visible
* Make shipping name fields required
* Make shipping address fields visible
* Make shipping address fields required

[Upgrade to the PRO version](https://www.southdevondigital.com/shop/wc-afour-pro/ "Upgrade to the PRO version") to unlock these extra options:

* Billing field prefix
* Make billing company field visible
* Make billing company field required
* Billing County/State Text
* Billing Postcode/Zip Code Text
* Make billing phone number field visible
* Make billing phone field required
* Shipping field prefix
* Make shipping company field visible
* Make shipping company field required
* Shipping County/State Text
* Shipping Postcode/Zip Code Text

You'll also get a year of updates & priority support directly from the developer.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Adjust WC AFOUR settings under WooCommerce -> Address Fields on User Registration
